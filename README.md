# Freshman News Portal

## Project Description
Freshman News Portal is a web application that allows college students to search and read the latest news articles and trending topics. The portal centralizes information in a single, easy-to-use interface, helping students stay informed about topics relevant to their campus life and academic interests
## Purpose 
- Help students stay informed with live news and trending topics
- Provide a foundation for a future Campus Life Super App
## Technologies Used
- HTML, semantic structure  
- CSS with responsive rules and layout techniques  
- Bootstrap for styling and responsive components   
- JavaScript for interactivity and event-driven programming  
- GNews API for fetching live news content  
- Accessibility features: semantic headings, alt text, color contrast  
## Instructions for setup in your local computer
1. Clone the project repository
2. Ensure you have a modern browser (Chrome, Edge, Firefox) to open HTML files  
3. Open `index.html` in your browser to start
4. For live news functionality, make sure you have an internet connection to access the GNews API
## Features 
- **Home Page:** Introduction  
- **News Search Page:** Dynamic search for news by keyword, displaying title, image, source, date, and link
- **About Page:** Details about the project
## How to Use
1. Navigate to the **News Search** page  
2. Enter a keyword in the search input 
3. Click "Search" to fetch the latest news articles
4. Browse the results, click "Read more" to open full articles in a new tab
