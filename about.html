About Freshman News Portal

This website application designed to help college students stay informed about the latest news and trending topics.

To search for news, go to the News Search page, enter your keywords, and click the "Search" button. The portal will display the most relevant news articles including title, image, source, and publication date.

Features

  • Live news search using the GNews API
  • Bootstrap for consistent design with custom CSS styling
  • Responsive layout for desktop, tablet, and mobile devices
  • Accessible design with semantic headings, alt text, and proper color contrast
